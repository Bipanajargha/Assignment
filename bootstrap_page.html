Bootstrap 5: Build with Power

Bootstrap is a popular open-source CSS framework that helps you build responsive, mobile-first websites quickly using prebuilt components and grid systems. 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css">

Use rows and columns to create layouts that scale across devices. 

<div class="row">
  <div class="col-md-6">Left</div>
  <div class="col-md-6">Right</div>
</div>

Bootstrap provides many button styles and alert components: 

<button class="btn btn-primary">Click Me</button>
<div class="alert alert-success">Success!</div>

Use cards to display content neatly with borders and padding: 

<div class="card">
  <div class="card-body">
    This is a Bootstrap card!
  </div>
</div>

Create a responsive navigation menu using the Navbar component: 

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <a class="navbar-brand" href="#">MySite</a>
</nav>
